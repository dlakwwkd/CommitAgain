실행방법
--
1. zip파일 받은 후 Release_v0.2 폴더 압축해제
2. 서버 실행(SkimaServer.exe)
3. 클라 2개 실행(Skima.exe)
4. 클라1 : 멀티플레이 -> 서버 접속 -> 방 생성, 
   클라2 : 멀티플레이 -> 서버 접속 -> 방 참여
5. 둘다 게임 시작을 누르면 자동으로 게임이 시작 됨
6. 마우스 우클릭으로 유닛을 이동시켜 볼 수 있음
